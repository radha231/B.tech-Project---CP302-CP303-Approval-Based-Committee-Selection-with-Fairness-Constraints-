# B.tech-Project---CP302-CP303-Approval-Based-Committee-Selection-with-Fairness-Constraints-
Fairness-Aware Multiwinner Voting is a computational social choice project implementing ILP, PAF-Comm, exact enumeration, and CIGAV for approval-based committee selection under fairness constraints. Evaluated on MovieLens and Goodreads to balance voter utility, diversity, and scalable fair representation.
